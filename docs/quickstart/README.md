
1. Download the reference data

we'll start with this smaller dataset for our quickstart guide. 
```
wget https://mondriantestdata.s3.amazonaws.com/mondrian-ref-20-22.tar.gz
tar -xvf mondrian-ref-20-22.tar.gz
```

```

```
